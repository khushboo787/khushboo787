### 🚀 Featured Projects

#### 💳 [Payment Wallet System](https://github.com/khushboo787/payment-wallet-system)
> Production-ready fintech backend — JWT auth, peer-to-peer transfers, Kafka notifications, Redis caching

- Built REST APIs for wallet management — deposit, withdraw, and transfer with **pessimistic locking** to prevent race conditions
- Integrated **Apache Kafka** for async payment notifications and **Redis** with 5-min TTL for balance caching
- Containerized using **Docker Compose** orchestrating 6 services: App · MySQL · Redis · Kafka · Zookeeper · Kafka UI
- Implemented atomic **@Transactional** transfers with automatic rollback on failure
- `Java 17` `Spring Boot 3` `MySQL` `Redis` `Kafka` `Docker` `JWT` `Swagger`
